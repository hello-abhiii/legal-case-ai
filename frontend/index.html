About LegalAI
Academic Project 2026–2027

This system acts as a decision-support tool for legal research, leveraging Machine Learning and Natural Language Processing to analyze court cases and extract actionable insights from historical precedents.

Core Technologies
  • Python & FastAPI
  • Scikit-learn (SVM/Random Forest)
  • TF-IDF & Cosine Similarity
  • Tailwind CSS & Vanilla JS
Performance

Achieves ~94.0% accuracy on cross-validation sets using a curated dataset of Indian court judgments spanning multiple IPC sections.

Disclaimer: This tool is experimental and designed for academic evaluation. It is not a substitute for professional legal counsel or human judicial decision-making.
